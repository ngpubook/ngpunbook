Проект "КНИГА ЗА КНИГУ"

Главная Контакты



В этой книге вы найдете все, что необходимо знать для создания отличных веб-сайтов. Начав с изучения принципов функционирования Интернета и веб-страниц,
к концу книги вы освоите приемы создания сложных сайтов, включая таблицы стилей CSS и графические файлы, и научитесь размещать страницы во Всемирной паутине.
Книга включает упражнения, с помощью которых вы освоите разнообразные техники работы с современными веб-стандартами (включая HTML5 и CSS3).

Вы научитесь:
• создавать HTML-страницы и добавлять на них текст, ссылки, изображения, таблицы и формы
• работать с каскадными таблицами стилей — форматировать текст, устанавливать цвета и фон, компоновать макеты страниц и создавать простые анимационные эффекты.
• использовать новые для HTML5 элементы, API-интерфейсы и свойства CSS3 при верстке веб-страниц.
• подготавливать страницы для отображения на мобильных устройствах посредством применения техник адаптивного веб-дизайна.
• ваботать с языком JavaScript, и разберетесь, почему он так важен для веб-дизайна.
• создавать и оптимизировать графические изображения для веб
