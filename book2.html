Аннотация к книге "Язык программирования Java SE 8. Подробное описание"
Эта книга написана разработчиками языка Java и является полным техническим справочником по этому языку программирования.
Она обеспечивает полный, точный и подробный охват всех аспектов языка программирования Java.
В ней полностью описаны новые возможности, добавленные в Java SE 8:

лямбда-выражения,
ссылки на методы,
методы по умолчанию,
аннотации типов и повторяющиеся аннотации.
В книгу также включено множество поясняющих примечаний.

В ней аккуратно обозначены отличия формальных правила языка от практического поведения компиляторов.
